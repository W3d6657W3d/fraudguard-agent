# Resume Notes

## English Version

Built a real-time fraud investigation Agent integrating transaction profiling, rule retrieval, risk feature analysis, and evidence-chain generation. Used tool calling and RAG retrieval to produce automated risk reports and support human-in-the-loop review.

## Interview Talking Points

- Why PR-AUC is more suitable than ROC-AUC in highly imbalanced fraud scenarios.
- How real-time fraud detection differs from offline classification.
- How threshold selection reflects the trade-off between false positives and missed fraud.
- How Agent workflows can assist analysts without replacing final human review.
